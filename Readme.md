# World TV PWA - Live TV Streaming

A Progressive Web App for watching live TV channels worldwide with adaptive streaming quality.

## Features

- 📱 **PWA Support** - Install on your device home screen
- 🎥 **Live TV Streaming** - Watch channels from around the world
- 🔄 **Adaptive Quality** - Automatically adjusts based on network speed
- 📡 **8 Categories** - News, Sports, Entertainment, Movies, Music, Kids, Documentary, International
- 📶 **Offline Support** - Basic offline functionality
- ⚡ **Fast Loading** - Optimized for slow connections
- 🎨 **Material Design** - Clean and modern UI
- 📱 **Mobile Optimized** - Touch-friendly interface

## Installation

### For Users
1. Open the app URL in Chrome or any modern browser
2. Click "Install" button in the header
3. Add to home screen when prompted

#### Prerequisites
